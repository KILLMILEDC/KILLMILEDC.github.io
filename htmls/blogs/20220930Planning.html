2022-09-30 | Plan | EN

I am sure you are aware of the work I have done in the previous months. If you don't know, you can find them in the "Projects" and "Researches" pages. # The above-mentioned device inevitably needs programming, so it is necessary to write a program or software that can realize the function.
# It may also be necessary to design a specific size shell and some fixing structures for the device.

These are my expected plans for the next three months. If I finish them on time, you can find them on my website and Github (some or all of the codes involved may be protected by the applied works, but if you abide by the relevant agreements, you can use them for free).
