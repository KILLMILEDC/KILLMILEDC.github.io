The software is a file number automatic generation software, the user only needs to define the corresponding generation rules, the software can generate the corresponding number according to the rules, compared with the traditional number generation process, the software not only fast generation, but also low error rate.

This paper describes an executable program that can run under Microsoft Windows operating system, named "Automatic document numbering tool" (referred to as "software" for short), and its version number is "V1.0".
