The purpose of the software development is to provide a comprehensive service for the monitoring of robotic arms (or to monitor the working status of small robotic arms). The software is programmed in Java and is intended for the industry "robotic arm monitoring". The main function of the software is "to carry out information management of robotic arm upper monitoring, to complete the relevant data integration, to carry out data management of robotic arm upper monitoring, to complete robotic arm upper monitoring processing, to use the software to run automatically, to assist users in operation and processing, and to realize reasonable robotic arm upper monitoring". The software has the following technical features: "Cloud computing software, using a hybrid separation model, makes the design and system operation organic, by optimizing the processing of the data engine to complete the operation of the software.

This paper describes an executable program that can run under Microsoft Windows operating system, named "Robotic arm host computer visual monitoring system" (referred to as "software" for short), and its version number is "V1.0".
