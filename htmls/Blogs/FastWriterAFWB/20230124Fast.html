Fast Writer — a fast website builder

2023-01-24 | Detail

Date of first publication Key words
Jan 24,2022 software;website;html
Abstract
This paper introduces a visual program based on .NET and C#, which can realize the visual writing of html files and improve the work efficiency to a certain extent.

2023-01-24 V.alpha 0.1

During the process of building the site, I encountered some difficulties in writing the code, for example, writing a whole paragraph in one line makes it difficult to go through during inspection, and when the code is too long it becomes difficult to read, etc.

Therefore, I have written a visual program which relies on .NET and C# to build. Through this software, it is possible to modularize the html code writing, by specifying some prescribed text input boxes and simply typing characters in the corresponding text boxes, its possible to generate html files in a specific format, which can improve certain efficiency while ensuring the readability of the files.

To write with the software, you only need to create a new workbench by "File >> New". There are three categories in the "New" list (Project, Blog and Research), There are specific breakdowns within each category, and they have the same basic framework as my personal website, which you can learn more about by reading the source code of the website or reading the sample file.Depending on the input range defined by the visualization workbench, you can freely specify basic things like titles, release dates, etc.

The text input box on the right side of the workbench is used to enter the body of the text. It can be understood as a text input device where you simply enter the text that will be displayed on the page, and it can be automatically integrated into a string in html format. Based on the "Style.css" in the sample file, the text in this text box can be categorized as "Text" and "Note" (the purpose of the categorization is Text without any prefix will be recognized as "Text", while for non-textual text such as notes or comments, simply add the prefix "<NOTE>", i.e. as of an "Enter" (\r\n) before any text with the prefix "<NOTE>" will be marked as non-text.For the "Blog" category, it has a special prefix "<IMAGE>", which is used to add images.

After writing, just click the "Save As" (File>>Save As) button to generate an html file, and then click the "Exit" (File>>Exit) button to exit the program.
