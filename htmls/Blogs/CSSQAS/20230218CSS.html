When writing .css files, the files are often too large due to irregular writing and too much irrelevant content, which will take more time to read and often cause some problems in the actual use.

So, I have written a script software based on Python language. With this software, some unnecessary content (such as indentation characters, extra spaces, duplicate symbols, and blank lines) can be removed from the .css file to reduce the file size and increase readability.

The software is simple to use, just run the executable and select the .css file to be revised in the file selection dialog. When the "The work is done." message box pops up, it means the execution is successful.
